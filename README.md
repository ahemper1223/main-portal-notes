# main-portal-notes

## 项目简介

`main-portal-notes` 是一个用于归档和发布独立 HTML 页面的仓库。这些页面通常用于记录、展示或临时托管某些静态内容，不绑定任何特定域名或网站。仓库本身不提供任何外部链接或商业导向，仅作为结构化的页面集合存在。

## 目录说明

仓库根目录下包含以下内容：

- `index.html` – 主入口页面，用于索引所有已归档的 HTML 页面。
- `pages/` – 存放所有独立 HTML 文件的子目录。每个文件代表一个独立的页面，命名遵循简洁、无歧义的原则。
- `assets/` – 存放页面所需的静态资源，如样式表、图片、字体等。资源文件按类型或用途进一步细分。

## 页面归档说明

所有归档的 HTML 页面均为独立文件，不依赖于外部数据库或后端服务。页面内容可能涉及技术笔记、临时信息展示、原型演示等，不承诺长期可用性或内容更新。每个页面均以标准 HTML 格式书写，兼容主流浏览器。

归档页面遵循以下原则：

- 不包含任何跟踪脚本、广告或外部请求（除非页面本身明确需要）。
- 文件名和路径保持稳定，避免破坏已有引用。
- 页面内容不涉及敏感信息、个人数据或受版权保护的材料。

## 维护说明

本仓库由维护者不定期更新。新增页面会放入 `pages/` 目录，并在 `index.html` 中添加对应链接。若页面内容过时或不再适用，可能会被移除或标记为“已归档”。

如需提交页面或提出修改建议，请通过 GitHub Issue 或 Pull Request 联系维护者。提交前请确保：

- 页面为纯 HTML，不含恶意代码或外部依赖。
- 文件命名清晰，与内容相关。
- 不包含任何商业推广、诱导点击或违规内容。
